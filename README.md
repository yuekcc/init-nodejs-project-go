# init-nodejs-project

a simple tool for init a new nodejs(npm) project.

## install

```sh
$ go version
go version go1.17.2 linux/arm64
$ go install github.com/yuekcc/init-nodejs-project@latest
$ init-nodejs-project -h
$ # ...
```

## usage

```sh
$ mkdir project-dir
$ cd project-dir
$ init-nodejs-project -author=hello -license=ISC
$ npm i
$ # done
```
